# Front end to Compass/Cipher AI

This contains the code for the web frontend of Compass/Cipher AI

It contains a combo of test panel that allows the user to see all the saved AI created summaraies. It also easily allows the user to create new ones.

It contains the financial contributions overview page that is accessible in production on the Compass/Cipher AI Chrome extension. That link will appear on company overviews where the applicable information is present in the system.
